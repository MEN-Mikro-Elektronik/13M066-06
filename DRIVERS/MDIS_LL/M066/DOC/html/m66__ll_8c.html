void m66_ll (  void   ) 
 

This function shows the use of: M66_Init() M66_Write() M66_Read() M66_BlockWrite() M66_BlockRead() M66_Exit().

It initialize the module, switches channel 0 on, reads channel 31, writes a pattern to all channels reads back the pattern and deinitialize the module.
